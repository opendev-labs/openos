# MoltOS Tech Specs

## AI Configuration
- **Model**: NanoPi-1.5B (Quantized Q4_K_M)
- **Inference Server**: OpenM (Fork of Ollama)
- **Interface**: `pi` (Natural Language Shell)

## Optimization (Potato-Tier)
- **ZRAM**: Enabled with 1.5x compression ratio.
- **Kernel**: Optimized for low latency.
- **Desktop**: XFCE (Lean & Fast)

## Project Goals
- Universal AI Shell integration.
- Offline task automation via Spoon.
- Local-first knowledge base integration.
