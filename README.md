# MoltOS: AI-Native Sovereign Distribution

MoltOS is a forked adaptation of Kali Linux, optimized for low-spec hardware and privacy-first local AI operations.

## Core Stack
- **Engine**: OpenM (Telemetry-free)
- **Soul**: NanoPi (1.5B AI Model)
- **Agent**: Spoon (Node orchestrator)

## Design Principles
1. **Offline-First**: All AI inference happens locally.
2. **Sovereign**: No telemetry, no cloud dependencies.
3. **Resource Efficiency**: Optimized for 8GB RAM and i7 hardware.
